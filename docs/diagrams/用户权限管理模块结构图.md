## 3. 核心类图

```mermaid
classDiagram
    class User {
        +Integer userId
        +String name
        +String email
        +String password
        +String phone
        +String role
        +String status
        +LocalDateTime lastLoginTime
        +Boolean isDeleted
    }

    class Role {
        +Integer roleId
        +String roleCode
        +String roleName
        +String description
        +Boolean isSuperAdmin
        +Boolean isDeleted
    }

    class Permission {
        +Integer permissionId
        +String permissionName
        +String description
        +Boolean isDeleted
    }

    class ProjectMember {
        +Integer memberId
        +Integer projectId
        +Integer userId
        +String permissionLevel
        +String projectRole
        +String status
    }

    class UserRole {
        +Integer userRoleId
        +Integer userId
        +Integer roleId
    }

    User "1" -- "*" UserRole : has
    Role "1" -- "*" UserRole : has
    User "1" -- "*" ProjectMember : has
    ProjectMember "1" -- "1" Role : belongs to
```
